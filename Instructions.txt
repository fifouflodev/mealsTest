
Objective : build a react-native application around meal recipes
Goal:

· to quickly define the real developer level

· to understand how she/he proceed to work on project

Subject:

· using public API https://www.themealdb.com/api.php , create a little project and put it on Github/Bitbucket;

· main features :

o list of recipes with main info: photo, title, category;
o search bar on the top of the screen, the list should be updated with the result of search query;
o on click on one item, details screen should be displayed with more info about recipe: photo, title, category, and ingredients;

· note: design does not matter

Important:

this test will mostly evaluate the quality of code!
